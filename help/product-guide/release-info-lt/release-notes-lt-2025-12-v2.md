# Version Dezember 2025 von Produktschulungen und Lerninhalten

Dieser Versionshinweis behandelt die neuen und erweiterten Funktionen, die mit der Version vom Dezember 2025 der Produktschulungen und Lerninhalte eingeführt wurden. Darüber hinaus wurden alle gemeldeten Probleme und Fehler in dieser Version behoben, wodurch eine verbesserte Stabilität und Leistung gewährleistet wurde.

## Authoring

- **Neue Optionen im Menü Einfügen**: Einführung neuer Optionen im Menü Einfügen der Editor-Symbolleiste, um Lerninhalte zu bereichern:

   - **MathML-Gleichung**: Fügen Sie MathML-Gleichungen nahtlos für technische oder wissenschaftliche Themen ein.
   - **Wissenstest**: Fügen Sie innerhalb der Lernthemen schnelle, nicht bewertete Quize hinzu, um das Verständnis der Lernenden zu überprüfen.
   - **H5P**: Interaktive H5P-Pakete für ein verbessertes Lernerlebnis einbetten.

  Weitere Informationen finden Sie unter [Weitere Optionen im Menü Einfügen](../learning-content/lc-other-insert-options.md).

  ![](assets/knowledge-check-topic.png){width="350" align="left"}

- **Neue interaktive Widgets**: Sie können Lernende mit einigen neuen interaktiven Widgets ansprechen, die Inhalte eindrucksvoller gestalten: **Zum Anzeigen klicken**, **Karte wechseln** und **Tabulator**.

  Weitere Informationen finden Sie unter [Verwenden interaktiver Widgets](../learning-content/lc-widgets.md).

  ![](assets/new-widgets-options.png){width="350" align="left"}

- **Entsprechendes beachten**: Ein neuer Fragetyp, **Entsprechendes beachten**, ist für Quiz verfügbar. Lernende können Elemente aus zwei Listen abgleichen, um verwandte Ideen miteinander zu verbinden und kritisches Denken zu fördern.

  Weitere Informationen finden Sie unter [Quizfragetypen](../learning-content/quiz-insert-questions.md#question-types).

  ![](assets/question-types.png){width="350" align="left"}

## Überprüfung

- **Prüfungsaufgabe erstellen**: Sie können jetzt eine Prüfungsaufgabe für Ihren Lernkurs erstellen und sie dem Prüfer für sein Feedback zuweisen. Dies sichert die Qualität der Inhalte, optimiert die Zusammenarbeit und erleichtert das Tracking von Überprüfungen.

  Für Details siehe [Prüfungsaufgabe erstellen](../learning-content/manage-course.md#create-review-task).

  ![](assets/create-review-task-lc.png){width="350" align="left"}

## Content-Management

- **Wiederverwendbarer Inhalt**: Sie können vorhandene Inhalte in mehreren Kursen wiederverwenden. Diese Funktion sorgt für Konsistenz und reduziert Duplizierungen.

  Weitere Informationen finden Sie unter [Grundlegende Bausteine hinzufügen](../learning-content/lc-basic-blocks.md).

  ![](assets/reusable-content.png){width="350" align="left"}
