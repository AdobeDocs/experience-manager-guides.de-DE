>
> Für die Generierung der AEM-Site-Ausgabe in Version 5.0 muss die Veröffentlichungsinstanz von Adobe Experience Manager über Service Pack 22, 21, 20 verfügen.




**Unterstützter Browser**

- Google Chrome
